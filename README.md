# CHOTU 🔗

*A fast, lightweight, and simple URL shortener.*

---

### 🚀 Features

**Instant Redirects**  
Quickly routes users from short links to their original destinations.

**Storage Management**  
Automatically expires older/unused URLs (TTL) to manage database memory efficiently.

**Continuous Deployment**  
Seamlessly synced with GitHub for automatic updates.

---

### 🛠️ Tech Stack

**Hosting:** [Vercel](https://vercel.com/) | **Database:** [Upstash Redis](https://upstash.com/)
